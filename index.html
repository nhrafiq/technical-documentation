What is Java?

Java was created in 1995 and is owned by Oracle. It is used for

  • Mobile, desktop, and web apps
  • Web servers
  • Games
  • Database connections

Java is useful because:

  • It works on different operating systems
  • It is easy to learn
  • It is popular
Hello World

All Java applications start with a class name, which matches the file name.

Here is a basic Java file:

public class MyClass {
    public static void main(String[] args) {
        System.out.println("Hello World"); 
    } 
}

You can run a Java file from the command line:

javac MyClass.java 
java MyClass
Basic Syntax

Java is case-sensitive. The class name must match the name of the file itself.

Moreover, each program needs a main method, which looks like this:

public static void main(String[] args)

Every statement will end with a semicolon.

Comments

Comments are used to make code more readable, and are important to ensure that code is maintainable. Java supports two kinds of comments: single-line and multi-line. The single-line comment looks like this: 

// This line will not execute
System.out.println("This line will execute.");

Multi-line comments look like these: 

/* This line will not execute, 
and neither will this one! */ 
System.out.println("This line will execute.");
Variables

Java has multiple types of variables, which store different values of data. Some of these include:

  • String
  • int
  • float
  • char
  • boolean

Java variables can be declared by specifying their types and values:

type variable = value;

Adding the final keyword will ensure that the value of this variable won't change:

final int num = 15;
